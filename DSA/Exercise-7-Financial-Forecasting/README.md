# Exercise 7 - Financial Forecasting

## Objective

Implement a recursive algorithm to forecast future financial values based on a fixed annual growth rate.

## Recursion

Recursion is a programming technique where a method calls itself until a base condition is reached.

## Formula Used

Future Value = Present Value × (1 + Growth Rate)^Years

## Sample Input

* Present Value = 10000
* Growth Rate = 10%
* Years = 5

## Output

Future Value after 5 years = 16105.10

## Time Complexity

O(n)

The recursive method makes one recursive call for each year.

## Optimization

The recursive solution can be optimized by using the mathematical formula directly, which avoids multiple recursive calls and improves performance.
