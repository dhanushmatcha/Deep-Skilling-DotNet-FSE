# 🎓 Student Course Portal

A modern Angular 20 web application developed as part of the **Cognizant Digital Nurture 5.0 Angular Hands-On Program**. The application demonstrates Angular fundamentals, advanced concepts, state management with NgRx, HTTP communication, routing, lazy loading, guards, interceptors, and unit testing.

---

## 📌 Project Overview

The **Student Course Portal** allows students to browse courses, view course details, enroll in courses, manage their profile, and interact with a responsive user interface backed by HTTP APIs.

This project was built progressively through **10 hands-on exercises**, covering beginner to advanced Angular development concepts.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/dhanushmatcha/Deep-Skilling-DotNet-FSE.git
```

Navigate to the project:

```bash
cd Deep-Skilling-DotNet-FSE/Angular/student-course-portal
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
ng serve
```

Open your browser:

```
http://localhost:4200
```

---

# 🌐 Mock Backend

This project uses **JSON Server** as a mock backend.

Start JSON Server:

```bash
json-server --watch db.json --port 3000
```

API Base URL

```
http://localhost:3000
```
