# EF Core 8.0 – Lab 4: Inserting Initial Data into the Database

## Overview

This lab demonstrates how to insert records into SQL Server using Entity Framework Core asynchronous methods.

---

## Objective

- Insert categories
- Insert products
- Save data asynchronously
- Verify inserted records in SQL Server

---

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Server Express
- SSMS

---

## Records Inserted

### Categories

- Electronics
- Groceries

### Products

- Laptop
- Rice Bag

---

## EF Core Methods Used

- AddRangeAsync()
- SaveChangesAsync()

---

## Output

```
Data inserted successfully.
```
