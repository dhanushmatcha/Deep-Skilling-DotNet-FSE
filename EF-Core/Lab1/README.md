
# EF Core 8.0 – Lab 1: Understanding ORM with a Retail Inventory System

## Objective

This lab introduces Object Relational Mapping (ORM) using Entity Framework Core. It demonstrates how EF Core bridges the gap between C# objects and SQL Server database tables.

## Concepts Covered

- Object Relational Mapping (ORM)
- Entity Framework Core
- EF Core vs Entity Framework
- EF Core Features
- SQL Server Integration

## Project Setup

- Created a .NET Console Application
- Installed Entity Framework Core packages
- Configured SQL Server provider
- Verified successful project build

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Server
- Visual Studio 2022

## NuGet Packages
