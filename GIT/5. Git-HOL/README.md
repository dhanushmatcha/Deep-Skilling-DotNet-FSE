# Git Hands-On Lab 5 - Clean Up and Push to Remote Repository

## Overview

This project demonstrates how to synchronize a local Git repository with a remote GitHub repository by verifying the repository status, listing branches, pulling the latest changes, pushing local commits, and confirming the updates on GitHub.

---

## Objectives

- Verify the repository is in a clean state
- List local and remote branches
- Pull the latest changes from the remote repository
- Push local changes to GitHub
- Verify successful synchronization

---

## Technologies Used

- Git
- Git Bash
- GitHub

---

## Git Commands Used

### Verify Repository Status

```bash
git status
```

### List Available Branches

```bash
git branch -a
```

### Pull Latest Changes

```bash
git pull origin master
```

### Push Local Changes

```bash
git push origin master
```

---

## Output

- Verified the master branch was clean.
- Listed all available branches.
- Pulled the latest changes from the remote repository.
- Pushed local changes successfully.
- Confirmed updates on GitHub.
